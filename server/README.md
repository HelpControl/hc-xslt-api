# XSLT API service

[https://github.com/HelpControl/hc-xslt-api](https://github.com/HelpControl/hc-xslt-api)

## Server

- Java
- maven
- Jetty

### Development

```
$git clone https://github.com/HelpControl/hc-xslt-api

$cd ./hc-xslt-api/server
$mvn compile 
```

#### Run Web Application

```bash
$mvn jetty:run -Djetty.port=8181
```
